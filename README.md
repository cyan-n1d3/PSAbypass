Bypass psa.wf, solve captcha manually, let userscript handle the rest.

- **starkroboticsfrc.com**: (Formerly *cashgrowth.online*)
- **exeygo.com / exe-links.com**: (Formerly *exe.io*)

- **shrinkme.click / themezon.net / mrproblogger.com**
